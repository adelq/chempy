ChemPy¶

ChemPy is a python package useful for (physical) chemistry. Currently it includes:

  • water density
  • water permittivity
  • Debye-Hückel expressions
  • Solver for equilibria
  • Integrated rate expressions (and convenience fitting routines)
  • Numerical integration routines for chemical kinetics (ODE solver frontend)

Units¶

Use of quantities is assumed, but only following attributes are actually accessed (parenthesis optional):

  • Kelvin
  • Joule
  • bar
  • meter
  • kilogram
  • mol
  • (nanometer)
